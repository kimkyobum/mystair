 mystair
"매일의 작은 체크리스트가 캘린더에 쌓여, 완벽한 STAR 자소서가 되는 맞춤형 취업 계단"** > **MyStair**는 마이스터고·특성화고 학생들을 위해 AI 기반의 데일리 목표 분해, 캘린더 경험 로깅, 그리고 STAR 기법 자소서 자동 변환 기능을 제공하는 올인원 커리어 포트폴리오 플랫폼입니다.


📌 개요 (Overview)

마이스터고 학생들은 3년이라는 짧은 기간 내에 실무 기술 습득, 자격증 취득, 현장 실습을 완수해야 합니다. 하지만 **실습 경험의 빠른 휘발**과 **장기 목표 실행력 부족**으로 인해 3학년 취업 시즌 시 자기소개서 작성과 진로 선택에 어려움을 겪습니다.

**MyStair**는 이러한 문제를 해결하기 위해 [목표 분해 ➔ 캘린더 로깅 ➔ AI 자소서 자동 완성]으로 이어지는 선순환 구조를 제공합니다.

✨ 핵심 기능 (Key Features)

1. 🎯 AI 목표 분해 데일리 체크리스트 (Micro-Task Engine)

- *Big Goal ➔ Small Action*: 큰 목표(예: 전기기능사 취득, 설비보전직 서류 준비)를 입력하면 AI가 오늘 당장 실천 가능한 15분 단위 체크리스트로 분해
- *스트릭 & 랭킹*: Daily 미션 완료 시 연속 달성일(Streak) 유지 및 교내/전공별 랭킹 포인트 부여

2. 📅 캘린더 기반 경험 데이터베이스 (Experience Calendar)

- *경험 자동 적재*: 완료된 데일리 체크리스트와 실습 스냅샷이 캘린더 타임라인에 자동 저장
- *스마트 태깅*: #PLC, #회로점검, #CAD설계, #팀워크 등 직무/역량 키워드 자동 분류

3. 🤖 AI 기반 STAR 자소서 자동 추출 엔진 (Experience Extractor)

- *데이터 선택*: 캘린더에서 자소서 소재로 사용할 특정 기간이나 태그 선택
- *STAR 기법 자동화*: 비정형 실습 기록을 **Situation(상황
