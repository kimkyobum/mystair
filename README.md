# mystair
마이스터고 학생들의 현실은 밖에서 생각하는 것보다 훨씬 숨 가쁘게 흘러갑니다. 일반 고등학교 학생들이 수능을 준비하는 3년이라는 시간 동안, 우리는 전공에 맞는 실무 기술을 익히는 것은 물론이고 필수 자격증 취득과 현장 실습까지 완벽하게 완수해야만 합니다. 문제는 이렇게 하루하루 치열하게 배우고 경험하는 소중한 실습 내용들이 너무 빠르게 휘발된다는 점입니다. 당장 눈앞의 과제들을 쳐내다 보면 장기적인 목표를 세우고 꾸준히 실행할 여력이 부족해지고, 결국 가장 중요한 3학년 취업 시즌이 닥쳤을 때 자기소개서에 어떤 내용을 적어야 할지, 심지어 내 적성에 맞는 진로가 무엇인지조차 막막해하는 친구들이 정말 많습니다.

저희 팀은 이런 구미전자공고 친구들의 현실적인 고민을 해결하기 위해 'MyStair'라는 서비스를 기획했습니다. MyStair의 핵심은 복잡하고 막막한 취업 준비 과정을 '목표 분해, 캘린더 기록, AI 자소서 완성'이라는 아주 자연스러운 선순환 구조로 만들어주는 것입니다.

가장 먼저, 학생들이 취업을 위한 큰 목표를 입력하면 AI가 똑똑한 페이스메이커로 개입합니다. 예를 들어 '전기기능사 취득'이나 '대기업 설비보전직 서류 통과' 같은 다소 막연하고 덩치가 큰 목표를 던져주면, AI가 이것을 오늘 당장 실천할 수 있는 15분 단위의 아주 작은 데일리 체크리스트로 잘게 쪼개줍니다. 여기에 매일 미션을 달성할 때마다 연속 달성일, 즉 스트릭(Streak)이 쌓이고 교내나 전공별로 랭킹 포인트가 부여되는 요소를 더해 지루하지 않게 목표를 향해 나아갈 수 있도록 돕습니다.

마지막으로 취업을 코앞에 둔 시점이 되면, 그동안 쌓아둔 이 데이터들이 가장 강력한 무기가 됩니다. 캘린더에서 자소서 소재로 쓰고 싶은 특정 기간이나 태그를 선택하기만 하면, AI 기반의 자소서 자동 추출 엔진이 작동합니다. 파편화되어 있던 비정형적인 일상 실습 기록들을 AI가 깊이 있게 분석해서, 기업 인사담당자들이 가장 선호하는 STAR(상황, 과제, 행동, 결과) 기법에 맞춰 논리적이고 완성도 높은 자기소개서 초안으로 자동 변환해 주는 것이죠.

MyStair는 단순히 일상을 기록하는 다이어리를 넘어, 치열했던 마이스터고 학생들의 3년을 가장 완벽한 취업 포트폴리오로 탈바꿈시켜 주는 강력한 무기가 될 것입니다.
